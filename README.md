# About the project

### From fan to fan, this is a web site for search cards the universe Yu-Gi-Oh!, fully responsive website, made with React consuming API from https://ygoprodeck.com/api-guide/ .

### Link: https://yugipedia.netlify.app/ .

# How to use?

## The purpose of this project is to facilitate the search for letters through the following filters:

## 1- Full card details

### If you click on the "details" button, it will load an expanded version of the card containing all possible information.

![Full details](https://github.com/thevalter/yugipedia-react/blob/main/src/assets/readme/fullCard.png)

## 2- Looking for a single card

### Just type in the search box and choose the card with the name you are looking for.

![Search card id](https://github.com/thevalter/yugipedia-react/blob/main/src/assets/readme/searchCard.gif)

## 3 - Looking for archetypes

![Search archetype](https://github.com/thevalter/yugipedia-react/blob/main/src/assets/readme/archetype.png)

## 4 - Looking for types

![Search type](https://github.com/thevalter/yugipedia-react/blob/main/src/assets/readme/type.png)

## 5 - Looking for race

![Search race](https://github.com/thevalter/yugipedia-react/blob/main/src/assets/readme/race.png)

## 6 - Looking for level

![Search level](https://github.com/thevalter/yugipedia-react/blob/main/src/assets/readme/level.png)

## 7 - Looking for link

![Search link](https://github.com/thevalter/yugipedia-react/blob/main/src/assets/readme/link.png)

## 8 - Cards filtering

### Just type in the filter box and it will automatically return the cards that contain the name.

![Filter card](https://github.com/thevalter/yugipedia-react/blob/main/src/assets/readme/filterCards.gif)
